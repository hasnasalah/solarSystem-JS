# solar system Practice

## Overview
This project has two parts JavaScript exercise.  
It uses **arrays**, **objects**, and **conditional logic**.

## features
The code demonstrates:
- Accessing and modifying nested Arrays and nested objects.
- Adding and updating values in objects.
- Performing calculations with object data.
- Type checking and data correction using 'typeof'.
